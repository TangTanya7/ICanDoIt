# ICanDoIt · 我能行

## 做了什么

ICanDoIt 是一个基于行为心理学的**目标达成平台**。它不生产知识内容，而是解决一个更本质的问题：**用户明明知道该做什么，却始终坚持不下来。**

产品通过四大核心机制——AI 规划导师、无阻力触达、即时反馈、温暖伙伴——将"想做"变成"真的在做"，让坚持自然发生。

## 谁在使用

18-35 岁的自我提升者：学生、职场新人、斜杠青年。

典型用户心声：
- "收藏了一堆 B 站课程，一个没看完"
- "我缺的不是方法和资源，是持续行动的动力"
- "开头很兴奋，过一周就忘了"

## 解决什么问题

| 用户痛点 | ICanDoIt 的破解方式 |
|---|---|
| 目标太大，无从下手 | AI 对话式拆解，3 个问题生成每日微任务 |
| 三天热度，动力消退 | 未来愿景激发 + 身份叙事反馈 + 多维进度可视化 |
| 忘记做，习惯断裂 | 锚点绑定日常习惯（刷牙后、午饭后），搭便车触发 |
| 做了看不到效果 | 任务完成 0.3s 内即时反馈 + 周/月进度回顾 |
| 断签一次就放弃 | 不惩罚、不归零、温暖复归弹窗 |
| 收藏了一堆资源从来不看 | 直接贴链接，AI 帮你拆成课时排进每日任务 |

## 如何完成一次完整操作

### 第一步：设定目标

用户进入 App，通过对话式引导回答 3 个问题（目标名称、计划周期、每天投入时间），并选择一个日常习惯作为锚点（如"起床后""刷牙后"），将新行为嫁接到旧习惯上。

也可以选择**直接粘贴学习链接**（如 B 站视频、TED 演讲、微信读书），系统自动提取内容标题，基于资料拆分课时。

### 第二步：AI 规划师自动生成方案

AI 规划师在 15 秒内完成：
1. 分析目标和现有水平
2. 设计技能成长路径
3. 拆分为 2-4 个学习阶段
4. 生成每日任务（学习+练习+反馈混合编排）
5. 从 B 站、TED、Coursera、微信读书等 10+ 平台匹配优质资源（或基于用户上传的资料拆分课时）
6. 生成验证练习题

每个任务都带有**一键直达的学习链接**，点击即跳转到对应平台。

### 第三步：每天完成 2-3 个微任务

今日任务页展示当天待办，每个任务卡片包含：
- 任务标题与描述
- 所属目标标签
- 预估用时（5-25 分钟）
- 一键跳转的资源链接（平台 + 标题）

勾选完成时弹出**轻量反馈 Toast**：
- 单任务完成："👍 搞定！今天 1/3"
- 全部完成："✨ 今天的任务全部完成！明天继续"
- 里程碑节点："🚀 50 个任务！已经超过 90% 的人了"

### 第四步：断签后温暖复归

如果用户几天没来，再次打开时不会看到"你失败了"，而是收到一个温暖的回归弹窗：

> "嘿！3 天没见了。我不打算问你去哪了，每个人都有自己的节奏。重要的是——你回来了。之前 28 天攒下的底子还在，一个都没丢。"

不归零、不惩罚、降低当日任务难度，帮用户重新开始。

## 用了哪些 AI 能力

| AI 能力 | 应用场景 | 实现方式 |
|---|---|---|
| **智能目标拆解** | 将一个大目标拆分为阶段→每日任务 | 基于目标类别 + 时长 + 用户水平，AI 生成分阶段路径，输入+输出+反馈三类任务混合编排 |
| **资源智能策展** | 从全网匹配最优学习资源 | 按目标类别从 B 站、TED、Coursera 等 10+ 平台筛选并排序资源 |
| **链接内容提取** | 用户粘贴链接后自动识别标题 | 后端 API 抓取页面 og:title / title 标签，提取关键词作为资源名称 |
| **课时智能拆分** | 将用户上传的资料拆分为课时任务 | 基于资源数量和阶段分布，自动分配学习-练习-总结任务 |
| **动态反馈文案** | 任务完成时的身份叙事反馈 | 基于完成数、连续天数、里程碑节点动态生成分层文案 |
| **伙伴对话引擎** | 全触点的温暖伙伴对话 | 基于关系深度（Day 1 → Day 100+）、时段、状态等 6 维变量生成个性化文案 |

## 心理学理论基础

产品设计建立在 8 个经实证验证的心理学理论之上：

- **微习惯理论**（B.J. Fogg）：任务足够小 → 能力门槛低 → "不可能不做"
- **心理对比 WOOP**（Oettingen）：先想象成功画面，再对比当前差距，目标达成率提升 2-3 倍
- **身份驱动**（James Clear / Oyserman）：从"我要学英语"变成"我是一个持续学习的人"
- **执行意图**（Gollwitzer）：将新行为绑定到已有习惯，绕过意志力直接触发
- **操作性条件反射**（Skinner）：固定反馈 + 偶尔惊喜，0.3s 内即时呈现
- **自我决定理论**（Deci & Ryan）：满足自主性、胜任感、归属感三大需求
- **自我慈悲**（Kristin Neff）：断签后宽容 > 自我批评，避免"管他呢效应"
- **损失厌恶**（Kahneman & Tversky）：连续天数本身就是不想失去的东西

## 技术栈

- **前端框架**：Next.js 16 (App Router) + React 19 + TypeScript
- **样式方案**：Tailwind CSS 4
- **构建工具**：Turbopack
- **包管理**：npm
- **平台**：响应式 Web（375×812 移动端优先）

## 功能模块

```
ICanDoIt/
├── Onboarding          # 首次引导（心理学原理展示 + 产品介绍）
├── CreateGoal          # 目标创建（AI 对话式 → 规划动画 → 方案预览）
│   ├── DetailStep      # 目标信息 + 锚点绑定 + 资源上传
│   ├── PlanningStep    # AI 规划动画 + 平台展示 / 资料拆分
│   └── PreviewStep     # 阶段预览 + 任务展开
├── TasksTab            # 今日任务（周/月日历 + 任务卡片）
│   ├── WeekCalendar    # 周视图 + 进度条 + 连续天数
│   ├── MonthCalendar   # 月视图 + 热力格子
│   ├── TaskCard        # 任务卡片（勾选 + 资源跳转）
│   └── TaskFeedbackToast  # 完成反馈（分层文案）
├── GoalsTab            # 目标列表 + 详情
│   ├── GoalCard        # 目标卡片（进度 + 阶段）
│   └── GoalDetail      # 目标详情（计划 / 任务 / 统计）
├── ComebackModal       # 断签温暖复归弹窗
└── BottomNav           # 底部导航（今日 / 目标 / 伙伴 / 我的）
```
